agentmesh 
Synchronize repository intelligence across AI coding runtimes

Usage: agentmesh [OPTIONS] [COMMAND]

Commands:
  status          Fast snapshot of current state
  scan            Report detected runtimes and entities without writing
  init            Set up AgentMesh in a repository
  install         Install sync hooks
  sync            Run a sync pass
  diff            Show what sync would change
  apply           Apply pending changes from a previous diff
  doctor          Run deep health checks
  restore         Restore a preserved runtime version
  ack             Acknowledge pending conflict resolution
  watch           Start the watcher daemon
  upgrade         Repin integrity to the current binary
  uninstall       Remove AgentMesh local wiring
  reconcile-lock  Recover from a textual lockfile merge conflict
  help            Print this message or the help of the given subcommand(s)

Options:
      --silent         Suppress informational output
      --no-color       Disable ANSI color output
      --color <COLOR>  Override color behavior [default: auto] [possible values: auto, always, never]
      --cwd <CWD>      Run as if invoked from this directory
  -v, --verbose...     Increase output detail
  -h, --help           Print help
  -V, --version        Print version

Usage: init [OPTIONS]

Options:
