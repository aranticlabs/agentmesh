Adapter ecosystem

Bundled adapters first, external adapters later.

AgentMesh v0.1 ships two first-party adapters compiled into the same binary. The external adapter trust model is reserved for v0.2 and later.

Bundled v0.1

Claude Code

Imports and emits:

  • CLAUDE.md
  • .claude/skills/<name>/SKILL.md
  • .claude/agents/<name>.md
  • .claude/settings.local.json hook entries
Bundled v0.1

Codex

Imports and emits:

  • AGENTS.md
  • .codex/skills/<name>/SKILL.md
  • .codex/agents/<name>.toml
  • .codex/hooks.json hook entries

External adapters

v0.2 introduces the adapter directory and third-party trust tiers. v0.1 keeps runtime parsing and emitting inside first-party Rust crates so the release can harden the core protocol before expanding the ecosystem.

AgentMesh adapters are part of the single static binary in v0.1.
