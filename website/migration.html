Migration

Versioned surfaces stay separate.

AgentMesh has independent versions for the artifact, adapter protocol, lockfile schema, and entity schemas. Upgrades fail loudly when a newer lockfile is not supported.

Fresh repositories

agentmesh scan
agentmesh init --yes --skip-hooks
agentmesh sync
agentmesh doctor

Existing Claude or Codex setup

Start with `scan` and `diff`. Initialize without hooks first if you want to inspect the canonical projection before installing runtime hooks.

Binary upgrades

agentmesh upgrade --dry-run
agentmesh upgrade --yes

Clean removal

agentmesh uninstall --yes
agentmesh uninstall --yes --purge
